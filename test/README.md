# Тесты и тестовые файлы

В эту папку рекомендуется помещать:

- Юнит-тесты для функций из helpers.js и FileToJsonNode.node.js
- Примеры файлов разных форматов (docx, xlsx, pdf, txt и др.) для проверки работы нода

Пример структуры:

- test/
  - files/
    - example.docx
    - example.xlsx
    - ...
  - unit/
    - helpers.test.js
    - node.test.js
